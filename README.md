# Evklid - Construction Project Management Landing Page

A high-fidelity, multi-section corporate landing page developed from a Figma mockup. This project focuses on professional CSS architecture, maintainable code, and semantic accessibility.

## Key Features:

* **BEM Methodology:** Implemented Block Element Modifier (BEM) naming convention for clean, modular, and scalable CSS.
* **Semantic HTML5:** Used descriptive markup to ensure better SEO and screen-reader accessibility.
* **Modern Layouts:** Built using a combination of **Flexbox** and **CSS Grid** for complex structural elements.
* **Interaction:** Features accessible interactive elements, including custom hover states and transitions.
